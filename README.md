# hCaptcha & 3DS Automation Solver Service

Puppeteer-based Express microservice for handling hCaptcha checkbox challenges and 3DS payment authentication flows.

## Features
- Express API server with CORS and Rate Limiting
- Puppeteer stealth plugin integration
- Headless Chrome detection evasions
- hCaptcha solving endpoint (`/api/solve-hcaptcha`)
- 3DS completion automation endpoint (`/api/3ds-automate`)
- Health check endpoint (`/health`)

## Installation

```bash
npm install
```

## Running

```bash
npm start
```
